### TB

#### Instruções

1. Descarregar o scheduler.py
2. Abrir o ficheiro
3. Correr o ficheiro
4. Inserir o número de eventos
5. Preencher os campos necessarios
6. Com o joystick, alternar entre eventos e com o clique do meio, expandir o evento e ativar o alarme
